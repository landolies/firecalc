Edit SSA tables

Verified through 2026 fact sheet

Bend points are computed from AWI using the SSA formula round($180 or $1,085 × AWI(year-2) / AWI(1977)), so you don't edit them directly — update the AWI table and bend points re-derive. Source: ssa.gov/oact/cola/bendpoints.html.
